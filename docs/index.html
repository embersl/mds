What's Modular Design System?

The Modular Design System is a collection of web modules created for various content and design strategies. Each module comprises components that may be shared with other modules. Together they provide a consistent, unified and adaptive visual presentation. With easy modification and customization, the system offers an efficient approach to common design tasks.

    Visual

  • Structure

    The headers and layout composition help establish the hierarchy. Let the logical and meaningful structure guide viewers through the content.

  • Unified look

    Balanced formatting & consistent appearance across all modules and components. Universal styles and CSS get put to work.

  • Minimal styling

    A clean wireframe feel allows for complete focus on concept & strategy without visual distractions. Visual customization comes later in the design process.

  • Responsive layout

    Modules are optimized for multiple viewport sizes to deliver best experience on desktop, tablet, mobile devices. Visual mockups are available for 3 viewport breakpoints.

  • Versatile

    It’s easy to mix & match, repurpose and reinvent based on the needs. Everything can be customized to suit the project’s content and strategy.

  • Functions

    The collection covers a wide range of components for common design tasks. Each module designed with clear purposes and usages in mind.

Collaboration

We often need to process the information we receive from the client. Prioritizing and rearranging the content are the necessary steps to form a compelling narrative for visual design. Within the creative team we can better allocate responsibilities and strategize design options fast. When working with the other teams, we can effectively communicate with examples, use cases and common vocabulary.

Content

Copywriters can use the modules to assist storytelling. Understanding the usage, pattern, rhythm and relationship among the content is beneficial - with structure, visual, word count in mind, it’s easier to optimize content and expedite revisions.

Prototype

Help stakeholders visualize the outcome and build realistic expectation, understand product experience and responsive behaviors, potentially get more involved and identify problems early.

Continuity

Use the design system as a source of inspiration, and a start point for more interesting and complex creations. A standardized workflow makes it easy to transfer & pick up projects among colleagues.

Interaction

  • Responsive

    Fluid design adapts to different screen properties and user input mechanisms on both desktop and mobile devices.

  • Web forms

    Html web form & inputs provide good interaction experience across broswers and platforms.

  • Micro-interactions

    Subtle changes and animations serve as hints for people who use mouse, keyboard or touch screen to browse the web.

  • Print styles

    Dedicated printer optimized css that makes printed pages come out fast, look great and stay authentic to the original web design.
