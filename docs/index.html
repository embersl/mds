What's Modular Design System?

The design system is a collection of web modules created for various content strategies and scenarios. Each module comprises smaller web components that may be shared with other modules. Together they provide a consistent, modular and adaptive visual presentation. With modifications and customizations, the design system offers an efficient approach to common design tasks.

Where do I find this MDS

A pdf reference contains categorized modules and their responsive design variations.

A companion html/css file gives real world experience in browsers. See demo at https://embersl.github.io/mds/

A example library provides some example usages.

    Visual

  • Minimal styling

    A clean wireframe feel allows for complete focus on concept, strategy and problem solving without visual distractions.

  • Functions

    The collection covers a wide range of components and usages for most common design tasks. Designed with clear structure and hierarchy in mind.

  • Consistent look

    Universal formatting & sizing & spacing across all modules and components. Visual customization(color scheme, fonts, imageries) made easy.

  • Responsive layout

    Modules are optimized for multiple screens and input mechanisms, such as desktop, tablet, mobile devices. Visual mockups available for all 3 viewport breakpoints

  • Versatile

    Find inspirations, mix & match, customize and repurpose the components to suit the projectâ€™s needs.

  • Editorial Styling

    Styled headers like H1~H5 help identify the outline & hierarchy. Let the logical and meaningful structure of the content guide viewers through the experience.

Collaboration

Team members can build common vocabulary for visual components and layouts. Within the creative team people can better allocate responsibilities and prototype design options fast. When working with marketing team we can potentially communicate better by providing examples and use cases before we start doing actual design.

Copywriter

Review the modules and identify the parts that will be populated with the copy such as headings, sub-heads, captions, quotes, body text etc. Understanding the pattern, repetition and relationship among the content is beneficial - with outline/ structure, visual arrangement, word counts in mind, itâ€™s easier to optimize the content and reduce rounds of changes.

Designer

Use the design system as a source of inspiration, and a start point for more interesting and complex creations.

Stakeholder

visualize the outcome and build realistic expectation, understand product experience and responsive behaviors, potentially get more involved in the team input process and identify problems early.

Interaction

  • Responsive

    Fluid design adapts to different screen properties and user input mechanisms on both desktop and mobile devices.

  • Web forms

    Html web form inputs provide good interaction experience across broswers and platforms.

  • Microinteractions

    Subtle changes and animations serve as hints for people who use mouse, keyboard or touch screen to browse the web.

  • Print styles

    Dedicated print optimization css that makes printed pages come out fast, look good and stay authentic to the original design.
